Features

Entirely Flexible Layout

Enjoy an entirely flexible layout, and never touch a grid-system again. Using flexie.js, you can now code in newly-introduced flexible box model. Now, instead of using float: left or float: right, all you need to do is declare a flex-order, and you are set to go! All this, in all browsers including IE6 and up!

Blazing Fast

Based on a source you know and trust, the HTML Boilerplate , Giganta takes from the learned lessons of the best developers. All scripts are minified upon deployment, CSS compressed and in LESS format, and most scripts loaded via CDN.

Loaded With Goodies

When it comes to content, Giganta makes getting your content out there as easy as possible. Not only does Giganta come with thousands upon thousands of lines of professional CSS3 designs and animations... it also includes every single sublime text 2 snippet you'll ever need!

Cross-Browser and Mobile-Friendly

Without having to know any jQUery Mobile code or any jQuery UI code, your site can be adapted for any type of screen. Using the Mobilize.js, your site is automatically transformed into a professional-looking mobile website.

Giganta

The world's best open-source HTML5 and CSS3 Boilerplate

With Giganta, creating professional and semantic HTML markup has never been easier. Whereas the HTML boilerplate is a minimal version of what you might need, Giganta strives to do the opposite. With hundreds of lines of possibly unnecessary CSS3 code, and semantic markup to help you along, in addition to Sublime Text 2 snippets, you can go from designing to marking up your content faster than ever before.
